# 彈射機器人 (Ricochet Robots)

這是一個包含多種版本的「彈射機器人 (Ricochet Robots)」遊戲專案。
彈射機器人是一款經典的解謎遊戲，玩家的目標是利用場地上的障礙物（牆壁）以及其他機器人，以最少的步數將特定顏色的「目標機器人」移動到指定位置。
請注意：機器人一旦開始移動，除非撞到牆壁或另一個機器人，否則不會停下來！

## 專案內容

本專案目錄下包含以下幾個不同的實作版本：
1. **Pygame 版本 (`ricochet_pygame.py`)**: 使用 Pygame 打造的 2D 遊戲版本，具備完整的視覺效果與遊玩體驗。（推薦從此版本開始遊玩）
2. **CLI 版本 (`ricochet_cli.py`)**: 透過終端機 (命令列) 遊玩的純文字版本。
3. **GUI 版本 (`ricochet_gui.py`)**: 簡易的視窗介面版本。
4. **Unity 版本 (`RicochetRobots/`)**: 使用 Unity 遊戲引擎開發的專案資料夾。

## 環境安裝

在執行 Python 版本前，請確保您的系統已安裝 Python 3，並安裝必要的套件 (以 Pygame 為例)：

```bash
pip install pygame
```

## 如何遊玩 (啟動 Pygame 版本)

1. 開啟終端機 (Terminal) 或命令提示字元 (CMD)。
2. 進入專案所在的資料夾。
3. 輸入以下指令啟動遊戲：

```bash
python ricochet_pygame.py
```

### 操作方式說明
- **選擇機器人**: 通常透過滑鼠點擊或是特定按鍵來選取你要移動的機器人。
- **移動**: 使用 **鍵盤方向鍵 (上、下、左、右)** 控制機器人的移動方向。
- **規則**: 機器人會直線衝刺，直到撞上牆壁或其他機器人才會停下。請善用這個特性來轉彎，找到通往目標的終極路徑！
